============
Installation
============


Stable release
--------------

To install pgsync, run this command in your terminal:

.. code-block:: console

    $ pip install pgsync

This is the preferred method to install pgsync, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/

.. _Github repo: https://github.com/toluaina/pgsync
.. _tarball: https://github.com/toluaina/pgsync/tarball/master
